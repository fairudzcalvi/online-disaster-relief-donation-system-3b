# BayanihanRelief - Online Disaster Relief Donation System

## Description

**BayanihanRelief** is a comprehensive web-based disaster donation management system that facilitates easy and secure donations for disaster relief efforts. The system features a modern public interface and a powerful admin panel for managing donations, donors, and distributions.

## Website Name

**BayanihanRelief**

## As of November 17, 2025

### Pages
- **Landing Page** - `landing.html`
- **Admin Dashboard** - `admin.html` (with modular admin panels)

### Features

#### Public Donation System
- **Streamlined Donation Process** - Guest donations without required user registration
- **Quick & Easy** - Simplified flow for emergency situations
- **Secure Transactions** - Protected against SQL injection and other vulnerabilities

#### Admin Management System
- **Comprehensive Dashboard** - Monitor donations, donors, and distributions
- **Donor Management** - Track and manage donor information
- **In-Kind Donations** - Manage non-monetary contributions
- **Organization Management** - Partner organization tracking
- **Reporting & Analytics** - Generate insights and transparency reports
- **Distribution Tracking** - Monitor relief distribution progress

#### Technical Features
- **Responsive Design** - Works on all devices
- **Modern UI/UX** - Smooth animations and intuitive interface
- **Modular Architecture** - Scalable and maintainable codebase
- **Secure Backend** - Protected data management
